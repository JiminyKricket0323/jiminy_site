Reflection of Digit 100

Before I even decided to become a Digit student , I talked to my professor. This is my third time challenging myself to learn coding after failing twice, one from my high school and one from my previous college. The most significant challenge to understand how everything just works 0 and 1 seems simple as it's just a two number but it was difficult to embrace it as it is. As I was a little lost at the beginning of the class when we were setting up GitHub, my uncertainty of the purpose of GitHub was the factor that slowed me down to truly explore all of the exciting HTML and CSS codes.

During the class, I was introduced to two website development methods, GitHub and PSU WordPress. As I got introduced to GitHub, now I can explain in my words that it's like a computing cloud system where authorized people can access your coding. Through access, your professor can edit and fix up any errors or mistakes from your codes and share them with classmates. On the other hand, PSU WordPress was something that I would consider a more simple way to start coding from basic by all of the templates that are already made. However, I found it's not fully helpful to make self-progress and self-improvement by not allowing users to write up codes in lines. As PSU WordPress doesn't have a cloud-kind of system where other people can have access to your codes of the sites, it makes it difficult for learners to fix any error that they probably face a lot of times, especially in the beginning.

During my journey to create my website "Continuexperience" I've got stuck with image file format issues. Although there were no errors found in my coding, a couple of the images that I put in for my index file were only showing up in my files, not on the GitHub site. Later, my professor and I found that it was a tiny matter of file format. I've used upper cases for some of the image files like JPG and PNG but some of them remain as lower cases. Because I'm using a Mac which is not GitHub, it is fully cooperative and there were some delivery issues from my image file to GitHub.

Both of the formats require you to interpret the HTML codings and RGBA colour to make it fancy. While using GitHub, using Git Bash to do all the commands like pull and push to make sure GitHub codes for your site is up to date was an extra part that was required.

In the future, I'm looking forward to learning about Python Programming as I wish to handle some big data. I've already looking up project materials from websites called Kaggle, however, I haven't learned any Python programming yet so there's not much I can do for now. I believe one day I will learn Python programming and some other data related programming languages to bring a useful and relevant result from a project that can contribute to our society.
